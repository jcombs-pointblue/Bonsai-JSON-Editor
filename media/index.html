See it in action

Tree View

Your JSON, organized

Bonsai renders every JSON file as a navigable tree. Values are color-coded — green for strings, blue for numbers, orange for booleans. Expand and collapse objects and arrays to focus on the data that matters. The toolbar shows the jq key path to your selection.

Source View

Raw JSON when you need it

Toggle to a full text editor to see formatted or minified JSON. Edit directly as source text — changes reparse live and sync back to the tree. Use Format and Minify to reshape the text in one click.

jq Tutorial

Learn by doing

New to jq? The built-in tutorial walks you through 40+ lessons — from basic field access to complex multi-step pipelines. Each lesson explains the concept, provides an editable query, and lets you run it against real sample data with a single click.
